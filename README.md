## Technologies
Project is created with:

* React-Router version: 5.0.1
* Ant Design version: 3.23.2

## Setup
To run this project, install it loccaly using yarn or npm, and navigate to the http://localhost:3000/:

```
$ yarn install
$ yarn dev-start
```
